# Realistic Snake Game 🐍

## Description
This is a smooth and realistic Snake Game developed using Python and Pygame.

## Features
- Smooth vector-based movement
- Dynamic snake growth
- Collision detection
- Animated tongue effect
- Score display system

## Requirements
- Python 3.x
- Pygame

## How to Run
1. Install pygame:
   pip install pygame
2. Run:
   python snake_game.py
